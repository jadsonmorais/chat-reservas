# Skills Locais

Esta pasta contém "skills" (habilidades customizadas) específicas para este projeto. Funciona da mesma forma que os skills globais, mas aplicáveis apenas neste repositório.

## Como criar um skill?

1. Crie uma pasta para a sua skill aqui (ex: `minha-skill`)
2. Crie um arquivo `SKILL.md` dentro dessa nova pasta (obrigatório).
3. Adicione o frontmatter ao `SKILL.md` e suas instruções abaixo:

```yaml
---
name: Nome da Skill
description: O que esta skill faz para este projeto
---
```

## Arquivos Suportados
Você também pode incluir scripts em uma subpasta `scripts/` ou templates em `resources/` que a inteligência artificial pode ler e rodar quando esta skill for solicitada.
